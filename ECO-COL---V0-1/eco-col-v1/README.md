# ECO-COL V1 - Sistema de Tele-Ecografía Local

Sistema de gestión y visualización de estudios DICOM de ultrasonido.

## Instalación Rápida

```bash
make build
make test
```

## Uso

```bash
# Nodo de adquisición
make dev-acquisition

# Nodo de lectura
make dev-reading
```

## Características

- ✅ 100% Local (sin cloud)
- ✅ Costo $0
- ✅ DICOM compliant
- ✅ P2P architecture
- ✅ 5 radiólogos concurrentes
